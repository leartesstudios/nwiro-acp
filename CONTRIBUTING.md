# Contributing to local-llm-acp

Thanks for your interest! `local-llm-acp` is a small Rust binary — an ACP-to-OpenAI
stdio shim. Contributions are welcome via pull request.

## Build

```bash
cargo build --release
```

`rustls`-only is a hard invariant — never add `native-tls` (it breaks
`aarch64-pc-windows-msvc` cross-compilation). See the note at the top of `Cargo.toml`.

## Tests & checks

The CI gates a pull request must pass (see `.github/workflows/ci.yml`):

```bash
cargo build --all-targets
cargo test --bins
LOCAL_LLM_USE_CONNECTOR=1 cargo test --bins golden   # golden-transcript snapshots
cargo deny check advisories licenses                  # supply-chain + license gate
```

Recommended locally (good hygiene, not gated in CI):

```bash
cargo clippy --all-targets
cargo fmt
```

**Do not hand-edit golden snapshots.** They are generated by `insta`; review changes
with `cargo insta review` and commit only the `.snap` updates that result from
intended behavior changes.

## Third-party licenses

`THIRD-PARTY-LICENSES.md` is generated from `Cargo.lock` by `cargo about`. If you add
or update a dependency, regenerate it and ensure `cargo deny check licenses` passes:

```bash
cargo about generate about.hbs -o THIRD-PARTY-LICENSES.md
```

A new dependency under a non-permissive license is blocked until its SPDX id is
reviewed and added to `deny.toml` + `about.toml`.

## Pull requests

- Keep changes focused and match the surrounding style.
- Note any user-visible behavior change in `CHANGELOG.md`.
- For release mechanics, see `RELEASING.md`.
